## 배운 점

### 다트(Dart) For문

~~~dart
void main() {
    for (int i = 0; i < 10; i++) {
        print(i);
    }
}
~~~

다트(Dart) 언어에서 For문은 위와 같이 C언어에서 사용하는 For문에 형태와 동일한
~~~
for (초기화; 조건; 증감) {
    반복 실행될 코드
}
~~~
위와 같은 구조를 가졌다
