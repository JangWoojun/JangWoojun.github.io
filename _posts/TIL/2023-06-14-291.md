## 배운 점

### 다트(Dart) While문

~~~dart
void main() {
    int count = 0;

    while (count < 5) {
        print('Count: $count');
        count++;
    }
}
~~~

다트(Dart) 언어에서 While문은 위와 같이 C언어에서 사용하는 While문에 형태와 동일한
~~~
while (조건) {
    실행 할 코드
}
~~~
위와 같은 구조를 가졌다
