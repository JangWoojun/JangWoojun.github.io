## 배운 점

### ### 다트(Dart) Do-While문

~~~dart
void main() {
  int i = 1;

  do {
    print(i);
    i++;
  } while (i <= 10);
}
~~~

다트에서 사용하는 Do-While문 또한 C언어에서 보던 형태와 같으면 조건을 나중에 확인하기에 조건이 거짓이여도 한번은 실행한다는 특징을 가졌다
